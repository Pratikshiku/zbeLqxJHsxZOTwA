## Online Mall System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/31ce385177694aaea4c96edcc8da880f/ghb20250917) 


### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`/ `mybatis`/`Redis`/ `jdk1.8`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`/`Element-ui`/`node14`, etc.
- Database: `MySQL`/ `mybatis`/`Redis`/

### 2. Related Software (Any one of the listed software can be used to run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Feature Description
The system is divided into (roles): regular shoppers and administrators.
Main features: Mall frontend (for users, administrators can also use), Mall backend management (only for administrators)

**Mall Frontend Features:**
1. Login
2. User Registration
3. Mall Homepage
4. Carousel Banner Navigation
5. Product Categories, Secondary Categories
6. Recommended Products
7. My Shopping Cart
8. My Orders
9. Administrator has: Backend Management
10. Order Payment

**Mall Backend Management:**
1. Login
2. Secure Logout
3. Dashboard
4. Frontend Module
5. User Management Module
6. File Management Module
7. Product Category Management
8. Carousel Management
9. Product Management
10. Order Management
11. Chart Management
12. Revenue Ranking
